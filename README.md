# 💰 Expense Tracker

A modern and responsive Expense Tracker web application designed to help users efficiently manage their personal finances. The application allows users to securely register, log in, record income and expenses, monitor their financial balance in real time, and generate a personalized financial report through an intuitive and user-friendly interface.

---

## 📌 Overview

Expense Tracker simplifies personal finance management by providing a secure platform to record daily financial transactions. It automatically calculates total income, expenses, and available balance while presenting users with a personalized financial summary and savings insights based on their spending patterns.

---

## ✨ Key Features

* 🔐 Secure user registration and login authentication.
* 👩 Personalized dashboard for every registered user.
* 💰 Add and manage income transactions.
* 💸 Add and manage expense transactions.
* 🗑️ Delete transactions with automatic balance updates.
* 📊 Automatic calculation of total income, expenses, and remaining balance.
* 📋 Dynamic financial report generated from user transactions.
* 📈 Personalized financial status and savings suggestions.
* 🟢🔴 Color-coded indicators for income, expenses, balance, and transaction types.
* 💾 Persistent user data using Browser Local Storage.
* 📱 Fully responsive interface for desktop and mobile devices.
